# Retail Sales SQL Analysis

This project explores transactional retail sales data using SQL, Python, pandas and SQLite inside Jupyter Notebook.

## Project Goals
- Analyse customer spending
- Identify top-performing products
- Investigate revenue trends
- Explore data quality issues
- Practice advanced SQL techniques

## Tools Used
- Python
- pandas
- SQLite
- Jupyter Notebook
- matplotlib

## Key SQL Skills Demonstrated
- GROUP BY
- Aggregations
- Filtering
- Date analysis
- CTEs
- Window Functions
- Ranking
- Running totals

## Key Findings
- Revenue is concentrated among top customers
- Certain products drive significant revenue
- The dataset contains missing customer identifiers
- Revenue trends fluctuate over time
